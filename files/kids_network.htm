Long-press only (3 seconds) Prevents accidental toggling — must hold button for 3 seconds

The plugin writes a hotplug script to /etc/hotplug.d/button/30-kids-wifi. Each press toggles wireless.kids_wifi.disabled and runs wifi reload — no reboot required.

Built-in Button Assignments

Only enable buttons you want to use for Kids WiFi. Avoid enabling Reset unless you are sure — it may interfere with factory reset on some routers.

BTN_0 — Hardware Slider GL.iNet and similar routers. Slide ON = WiFi on, slide OFF = WiFi off.
WPS Button Single press toggles Kids WiFi. Available on most consumer routers.
Reset Button ⚠ Single press toggles Kids WiFi. Will prevent factory reset while enabled. Use with caution.
